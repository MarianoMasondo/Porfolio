# Portfolio Personal - Mariano Masondo

Este repositorio contiene mi portfolio personal, un sitio web creado para presentar quién soy, mi recorrido como desarrollador y algunos de los proyectos que fui construyendo durante mi formación y práctica profesional.

La idea principal del portfolio es mostrar mi perfil de una forma simple, clara y cercana: qué aprendí, qué tecnologías fui incorporando, qué proyectos desarrollé y cómo contactarme.

---

## Sobre el portfolio

El sitio está pensado como una carta de presentación profesional.  
Incluye información sobre mi formación, mis habilidades, mis idiomas, una breve presentación personal y una selección de proyectos que reflejan distintas etapas de mi aprendizaje.

Más que ser solamente una página técnica, este portfolio busca mostrar mi evolución, mi curiosidad por la tecnología y mis ganas de seguir creciendo dentro del desarrollo web.

---

## Secciones principales

- **Inicio:** presentación personal y breve descripción de mi perfil.
- **Educación:** formación como Full Stack Developer.
- **Skills:** habilidades técnicas, habilidades blandas e idiomas.
- **Proyectos:** selección de aplicaciones y sitios desarrollados.
- **Acerca de mí:** una mirada más personal sobre mi vínculo con la tecnología.
- **Contacto:** enlaces para comunicarse conmigo o conocer más sobre mi trabajo.

---

## Algunos proyectos incluidos

En el portfolio se pueden encontrar proyectos como:

- **Rick & Morty App:** aplicación para explorar personajes, filtrarlos, ordenarlos y guardarlos como favoritos.
- **Drivers App:** aplicación full stack sobre corredores de automovilismo, con búsqueda, filtros y creación de nuevos drivers.
- **Las Encinas Boutique:** e-commerce desarrollado en equipo aplicando metodología Scrum.
- **Weather App:** aplicación para consultar el clima de distintas ciudades.
- **Tienda Muebles:** sitio web orientado a practicar diseño responsive y maquetación.
- **Meeti App:** práctica de frontend con Bootstrap y SASS.
- **Podcast App:** sitio desarrollado para seguir fortaleciendo habilidades de maquetación y estilos.

---

## Objetivo del proyecto

El objetivo de este portfolio es reunir en un solo lugar mi perfil profesional, mi experiencia de aprendizaje y mis proyectos más representativos.

También funciona como una forma de seguir practicando, mejorando el diseño, ajustando detalles visuales y manteniendo actualizado mi camino como desarrollador.

---

## Tecnologías utilizadas

El proyecto fue construido principalmente con:

- HTML
- CSS
- JavaScript
- Vite

También incluye recursos visuales, estilos personalizados y pequeñas interacciones para mejorar la experiencia de navegación.
